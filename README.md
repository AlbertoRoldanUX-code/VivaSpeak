# VivaSpeak

Plataforma de agentes de voz IA para automatizar llamadas, WhatsApp y correo electrónico.

## 🚀 Tecnologías

- **Astro** - Framework web moderno
- **HTML/CSS/JavaScript** - Frontend
- **Deepgram** - Speech-to-text (STT)
- **GPT-4o mini** - Modelo de lenguaje (LLM)
- **ElevenLabs** - Text-to-speech (TTS)

## 📦 Instalación

```sh
npm install
```

## 🛠️ Comandos

| Comando           | Acción                                        |
| :---------------- | :-------------------------------------------- |
| `npm install`     | Instala las dependencias                      |
| `npm run dev`     | Inicia el servidor local en `localhost:4321`  |
| `npm run build`   | Compila el sitio para producción en `./dist/` |
| `npm run preview` | Previsualiza la build localmente              |

## 📂 Estructura del proyecto

```text
/
├── public/          # Archivos estáticos (imágenes, fuentes, etc.)
├── src/
│   ├── components/  # Componentes reutilizables (Header, Footer)
│   ├── layouts/     # Layouts de página
│   └── pages/       # Páginas del sitio (index.astro, etc.)
└── package.json
```
